Environments

Base URLs

Players Health provide you two different environments as shown in the following table. The base URL differentiate the environment between the two.

Environment URL

Sandbox

https://api-staging-athletesafety.playershealth.com/v1/

Production

https://api.athletesafety.playershealth.com/v1/

Use the Sandbox base URL to make requests to the sandbox environment and the Production base URL to make requests to the real Players Health Environment.

You need an API key to access the APIs if you don't have one, refer to Authentication section for more details.

Sandbox

You can experiment with our APIs in the Players Health Sandbox, which is self-contained and has no interaction in the real world. You can test the underlying services as necessary.
